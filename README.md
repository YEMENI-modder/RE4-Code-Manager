# **RE4 Code Manager**

## **تعريف الأداة**

أداة **RE4 Code Manager** توفر لك أكثر من **100 كود** جاهز،  
مع إمكانية تشغيلها بسهولة شديدة من خلال **واجهة بسيطة وخيارات مباشرة**.

---

## **مميزات الأداة**

- **أكثر من 100 كود جاهز للاستخدام.**
- **إمكانية إضافة أكواد مخصصة** من قبلك.
- **إضافة مجموعة أكواد دفعة واحدة** بسهولة.
- **تنظيم الأكواد داخل أقسام** حسب النوع.
- **دعم لغتين (العربية والإنجليزية)**  
  *(ملاحظة: النسخة العربية قد تحتوي على بعض المشاكل حاليًا).*
- **تشغيل أو إيقاف الأكواد بضغطة زر.**
- **تفعيل عدة أكواد دفعة واحدة** عبر خيار **Apply Selected**.
- **إمكانية استعادة الإعدادات الافتراضية** من داخل الإعدادات.
- **مقارنة بين أكثر من ملف EXE**.
- **نظام ذكي للاعتماد بين الأكواد**  
  (بعض الأكواد لن تعمل إلا بوجود أكواد أخرى، والأداة تدير ذلك تلقائيًا).
- **والمزيد...**
